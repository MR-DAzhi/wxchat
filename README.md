# 微信对话生成器

微信对话生成器是一款在线微信聊天对话制作工具。它支持设置苹果或安卓状态栏（包括手机电量、手机时间等），还能模拟不同用户角色，发送文字、语音、红包、转账等多种消息，堪称一款娱乐制作神器。

---

## 功能介绍

- **状态栏设置**：选择苹果或安卓状态栏，支持显示手机电量、手机时间等信息。
- **角色定制**：自定义不同的聊天角色，打造个性化对话场景。
- **多种消息形式**：支持发送文字、语音、红包、转账等多种消息类型。
